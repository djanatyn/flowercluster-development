zubkoland

hello!

zubkoland is a server for learning and building.

zubkoland runs on ansible; the configuration is available on github.

currently running services are:

  • mail (postfix 2.11.3)
  • web (apache 2.4.10)
  • irc (ngircd 21.1)

direct inquiries to djanatyn@zubkoland.org.
